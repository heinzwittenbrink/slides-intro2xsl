Changes

XSLT 3.0

The primary purpose of the changes in this version of the language is to enable transformations to be performed in streaming mode, where neither the source document nor the result document is ever held in memory in its entirety. Another important aim is to improve the modularity of large stylesheets… XSL Transformations (XSLT) Version 3.0
